Global Game Jam 2020

"Fight against your friends in this splitscreen arcade-style game! Sell the resources you mine and buy a variety of weapons to explode, shoot and push your enemies to their death!"

A 4-player game in which you need to be the last man standing. Each player has their own gravity so be careful where you step!

This project won us first prize at the University of Lincoln Jam Site!

Images

Development

This game was created for Global Game Jam 2020. Based on the theme of 'Repair'.

Going in to this game jam as a pair we knew what we wanted to make. We just needed a way to make the game fit the theme of the game jam. We had laid out our plans for the game jam on a whiteboard so that we could tick off the tasks as we went. This proved helpful and we ended up reaching most of the goals we set out for ourselves.

For this game I did the bulk of the programming, though some of the elements such as the lobby system were programmed by my partner.

Description

The goal for this game is to be the last player standing. The first part of the game requires the player to use their pickaxe to collect resources, which they can then sell. with the money they can visit the shop and buy weapons.

At this point in the game the players must try to kill each other, however with each player having their own personal gravity this can prove difficult. Once there is one player left the round is over.

The game ends once one player has won four rounds, having successfully built their rocket they can escape the doomed planet.

How To Play

The game requires the use of multiple controllers.

Controller
A - Jump
X - Pickup Item
Left Trigger - Interact
Right Trigger - Shoot / Mine
Left Stick - Move
Right Stick - Aim
